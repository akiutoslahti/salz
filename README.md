# SALZ - Suffix array based Lempel-Ziv data compressor

## Prerequisites

* Git
* CMake
  * Tested with version 3.15.3.
* GCC, or
  * Tested with version 9.3.0
* Clang
  * tested with version 10.0.0

## Build

```
git clone --recurse-submodules https://github.com/akiutoslahti/salz.git
cd salz
mkdir build
cd build
cmake ..
make
```
